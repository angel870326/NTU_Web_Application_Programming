# Guide

Step 1: 在終端機執行以下程式碼：

```
cd PATH
yarn create react-app YOUR_APP_NAME
cd YOUR_APP_NAME
yarn add bootstrap
yarn add typed.js
yarn add axios jquery
```

Step 2: 把 public 和 src 資料夾拉進去 YOUR_APP_NAME，直接取代即可。<br>

Step 3: 最後在終端機執行：

```
yarn start
```
